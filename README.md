## 🏛️ Overview
**ART-KEEP** is an advanced telematic system designed to modernize museum management. It integrates IoT, autonomous robotics, and cognitive computing to address three pillars of modern curatorship:
1. **Preventive Conservation:** Smart monitoring of environmental variables.
2. **Security & Tracking:** Real-time asset protection via RFID.
3. **Interactive Guidance:** Autonomous robotic assistance and crowd control.

## 🚀 Key Features
- **Context-Aware Monitoring:** Real-time analysis of temperature, humidity, and CO2.
- **Knowledge-Based Reasoning:** Automatically adjusts HVAC systems based on the specific material needs of each artwork (e.g., canvas, wood, metal).
- **Autonomous Navigation:** Robotic guides with dynamic **Motion Planning** to avoid overcrowding.
- **Security Protocols:** Immediate network alerts and room lockdown upon unauthorized artwork movement.

## 🛠️ Tech Stack & Topics
This project implements core concepts from the **Advanced Telematic Applications (ATA)** curriculum:
- **Cognition:** Action Planning, Knowledge Representation, and Reasoning.
- **Perception:** Sensing and Interpretation of environmental data.
- **Navigation:** Mapping and Localization for the robotic agent.
- **Networking:** Protocols for cooperating agents and distributed control.
